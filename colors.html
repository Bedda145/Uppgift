CSS Colors

Vad: styr färgen på text, bakgrunder, ramar och andra element. Hur: ange färger med namn (red), HEX-koder (#ff0000), RGB (rgb(255, 0, 0)) eller RGBA för transparens (rgba(255, 0, 0, 0.5)). Gör: ger designen liv och skapar visuell hierarki och stämning.

h1 { color: blue; } .box { background-color: #ff5733; } .element { border: 2px solid rgb(0, 255, 0); } .overlay { background-color: rgba(0, 0, 0, 0.5); /* Svart, 50% genomskinlig */ }
