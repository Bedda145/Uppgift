Display

Vad: Styr hur element beter sig i layout. Hur: block tar hela raden, inline ligger i textflödet och ignorerar width/height, inline-block ligger på rad men kan få width/height. Gör: påverkar radbrytning och storlek

.box { width: 100px; height: 100px; border: 1px solid #333; } .block { display: block; } .inline { display: inline; } .inlineblock { display: inline-block; }
