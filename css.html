Länk en css fil in i html filen

För att css ska kunna funka så måste man länka in det i html filen, det gör man med det här kommandot

<link rel="stylesheet" href="style.css">

Hur skriver man css

CSS är uppbyggt med selctor (regler) där man förklarar, vilka element som ska styleas och hur. Vi har då tre stycken begrepp, selector (vilket element), property (vad som ska ändras) samt value (till vad det ska ändra). Det ser ut såhär:

selector { property: value; }
