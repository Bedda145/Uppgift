Position

Vad: kontrollerar placering och kan bryta det vanliga flödet. Hur: static (standard), relative (flyttas från sin plats), absolute (placeras mot närmaste icke-statisk förälder), fixed (mot viewport), sticky (fastnar vid scroll.). Gör: positionerna med top/right/bottom/left i px/%.

.wrap{ position: relative; } .rel { position: relative; top: 10px; left: 10px; } .abs { position: absolute; top: 0; right: 0; } .fix { position: fixed; bottom: 10px; right: 10px; }
