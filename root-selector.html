root Selector

Vad: en selector för roten av dokumentet (samma som ´<html>´). Används främst för att skapa globala CSS-variabler. Hur: definiera variabler i :root med --variabel-namn: värde;. Använd dem sedan med var(--variabel-namn). Gör: gör det enkelt att återanvända värden och ändra dem på ett enda ställe.

:root { --primary-color: #0056b3; --main-font-size: 16px; } body { font-size: var(--main-font-size); } a { color: var(--primary-color); }
