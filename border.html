Vad: styr hur ett element visas och beter sig. Hur: block (tar full bredd, ny rad), inline (flyter med texten), inline-block (som inline, men med width/height). Gör: ändrar ett elements grundläggande layoutbeteende.

.element-1 { display: block; } .element-2 { display: inline; } .element-3 { display: inline-block; } }
